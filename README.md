# Poke-Manager


## Prerequisites

1. Node.js <https://nodejs.org/en/>

2. Git <https://git-scm.com/>

3. Python <https://www.python.org/downloads/>

## Setup

git clone https://github.com/eudai/poke-manager.git

cd poke-manager

npm install

npm start

visit <http://localhost:3000>

## Usage

Login

View Stats and IV

Transfer

Evolve

Rename

Favorite

Sort
